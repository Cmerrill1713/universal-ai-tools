```bash

```

### **3. Run GitLabs Setup** 🚀
```bash

```

- `POSTGRES_DB`: `universal_ai_tools`

- `API_KEYS`: (your encrypted API keys)
- `DEPLOY_TOKEN`: (your deploy token)
